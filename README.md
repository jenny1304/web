## 2. Độ ưu tiên trong CSS
  [:arrow_up:](#content-tables)
1. `!important ` = 10000
  2. `inline` = 1000
  3. `#id` = 100
  4. `.class` = 10
  5. `tag` (div, h1) = 1
  6. `*` (for all) = 0
